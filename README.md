# id-inator

## Running the script
Clone the repo
```
$ git clone https://github.com/MainKt/id-inator.git
```

Create and activate a venv
```
$ python -m venv .venv
$ . ./.venv/bin/activate
```

Install the dependencies
```
$ pip install -r requirements.txt
```

Run main.py
```
$ python ./main.py --help
```
